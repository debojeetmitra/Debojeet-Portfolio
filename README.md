# 🚀 Debojeet Mitra | Full-Stack AI/ML Developer Portfolio

A stunning, high-aesthetic portfolio built with **Next.js**, **Tailwind CSS v4**, and **DaisyUI v5**, featuring modern Glassmorphism, dynamic theming, and smooth animations.

## 🌟 Key Features

-   **🎨 Dynamic Theming**: Integrated theme switcher with support for 15+ DaisyUI themes (Coffee, Luxury, Valentine, etc.).
-   **💎 Glassmorphism UI**: High-end visual design with frosted glass effects, radial gradients, and animated backgrounds.
-   **⚡ Framer Motion Animations**: Smooth entry animations and micro-interactions for a premium user experience.
-   **📁 Project Showcase**: Detailed grid of 7+ professional projects with image previews, tech tags, and live links.
-   **📱 Fully Responsive**: Optimized for all device sizes, from mobile to ultra-wide displays.
-   **📊 Work & Education**: Professional timeline of educational history and key achievements (LeetCode, etc.).

## 🛠️ Tech Stack

-   **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & [DaisyUI v5](https://daisyui.com/)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/debojeetmitra/Debojeet-Portfolio.git
    cd Debojeet-Portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in browser**:
    Navigate to [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

-   `/src/app`: Next.js App Router and global styles.
-   `/src/components`: Reusable UI components (ThemeSwitcher, etc.).
-   `/src/data`: Data-driven content for projects and skills.
-   `/public`: Assets including project images and professional resume.
